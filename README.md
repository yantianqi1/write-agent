# 小说智能创作 Agent

一个能够自动创作长篇小说的智能 Agent 系统。

## 项目结构

```
小说agent/
├── src/                    # 源代码目录
│   ├── memory/             # 记忆系统模块
│   ├── story/              # 创作流程模块
│   ├── utils/              # 工具函数
│   └── api/                # API 服务
├── tests/                  # 测试目录
│   ├── unit/               # 单元测试
│   └── integration/        # 集成测试
├── docs/                   # 文档
├── config/                 # 配置文件
├── data/                   # 数据存储
├── 需求文档.md            # 项目需求文档
├── claude.md              # 开发进度跟踪
└── README.md              # 项目说明
```

## 核心功能

1. **一句话灵感输入** - 快速启动创作
2. **自动素材收集** - 检索数据库 + 联网搜索
3. **分层记忆系统** - 保证长篇一致性
4. **人机协作创作** - 用户审核 + 机器生成
5. **质量控制机制** - 多轮迭代优化

## 技术栈

- Python 3.10+
- FastAPI (API 服务)
- Chroma (向量数据库)
- LangChain (LLM 集成)
